# Brand Guide (Seed)

このフォルダは `agent_file_research` の file_search 用に起動時アップロードされる参照データです。

## トーン
- 断定しすぎない（根拠がある場合は引用URLを残す）
- 箇条書きは短く

## デザイン（スライドHTML）
- フォント: システムフォントのみ（例: Segoe UI / Arial / Helvetica）
- 画像: 外部URL禁止（http/https）・CDN禁止
- scriptタグ禁止
- 1スライド = `<section class="slide">`
- 余白（セーフマージン）を確保し、はみ出し禁止
